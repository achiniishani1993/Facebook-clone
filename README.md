# Facebook-clone
Recreate similar website like facebook to practice react skills

A modern Facebook-inspired UI clone built with React and TypeScript, featuring a dynamic feed, navigation bar, side panels, chat interface, and interactive components.

This project focuses on recreating the core layout and user experience of Facebook using clean component-based architecture.

🚀 Features

🧭 Navigation Bar
* Facebook-style top navigation
* Search bar for finding friends
* Notification icons (Friends, Messages, Settings)
* Profile image display

📰 Post Feed
* Add and display posts
* React to posts with:
👍 Like
❤️ Love
😮 Wow
* Dynamic like counter
* Add comments to posts
* Instant UI updates (no reload)

📂 Left Sidebar (Menu + Pages)
* Menu shortcuts:
* "Pages You Liked" section
🔍 Search/filter pages in real-time


💬 Right Sidebar (Contacts & Chat)
* Online friends list
* Click a user to open chat
* Chat popup UI:
* Header with user info
* Message input field
* Action icons (GIF, emoji, camera, etc.)
* Close chat functionality

🎯 Additional UI Sections

📢 Sponsored ad section
🎂 Birthday notifications
🟢 Online status indicators
